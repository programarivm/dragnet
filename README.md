## ReactNet

A real-time network traffic monitoring tool that uses a web interface with charts.

## 1. Server

### 1.1. `tcpdump` writer

    sudo php cli/tcpdump-writer.php eth0 $USER

### 1.2. WebSocket server

    php cli/ws-server.php

## 2. Client

### 2.1. Development mode

    npm start
